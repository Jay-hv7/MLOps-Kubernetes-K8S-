# MLOps-Kubernetes-K8S-
ML application deployment on Kubernetes

## What is Kubernetes
Kubernetes is open source system for managing containerized applications. It is a system for running and coordinating containerized applications across a cluster of machines. It manages life cycle of containerized applications.

Features:
- Load balancing : load distribution between containers
- Scaling 
- Resource management
- Self Healing
- Centralized monitoring and debugging
